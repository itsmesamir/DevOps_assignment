| Status Code | Meaning               |
| ----------- | --------------------- |
| 1. 200      | OK                    |
| 2. 201      | Created               |
| 3. 202      | Accepted              |
| 4. 400      | Bad Request           |
| 5. 401      | Unauthorized          |
| 6. 403      | Forbidden             |
| 7. 404      | Not Found             |
| 8. 405      | Method Not Allowed    |
| 9. 415      | Unsupported Media Type|
| 10. 500     | Internal Server Error |
| 11. 502     | Bad Gateway           |
| 12. 503     | Service Unavailable   |
